# Навигационная карта КСО

> Заполни по шаблону из `files/kso-flutter-research/UNDERSTANDING_KSO_FLUTTER_NEW.md` (п. 1.1)

* Меню (главный)
    * Детали товара (productId)
        * (кнопка «В корзину» — возврат в Меню)
    * Корзина
        * Тип заказа (takeaway / dine-in)
            * Способ оплаты (карта / наличные / СБП)
                * Процессинг → Успех / Ошибка
        * (можно вернуться в Меню)
    * [Нижняя панель корзины — с любого экрана]
