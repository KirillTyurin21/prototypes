# КСО Flutter — данные проекта

Здесь размещаются описания экранов, flow, модели данных для Flutter-приложения КСО (киоск самообслуживания).

## Структура

```
kso-flutter/
├── README.md              ← Этот файл
├── flow.md                ← Навигационная карта приложения
├── data-models.md         ← Модели данных (Product, Category, CartItem, Order и др.)
├── api-contract.md        ← API-контракт (если есть)
├── screens/
│   ├── 01_menu.md         ← Экран «Меню»
│   ├── 02_product.md      ← Экран «Детали товара»
│   ├── 03_cart.md         ← Экран «Корзина»
│   └── ...                ← По одному файлу на экран
└── docs/                  ← Сопроводительная документация (генерируется автоматически)
    ├── ARCHITECTURE.md
    ├── ROUTES.md
    ├── COMPONENTS.md
    ├── DATA_MODELS.md
    └── diagrams/
        ├── navigation-flow.puml
        ├── screen-states.puml
        ├── data-flow.puml
        └── architecture.puml
```
