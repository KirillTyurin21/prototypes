# Задача для бэкенда: Права доступа (Q3)

> **Дата:** 2026-06-11
> **От:** Кирилл
> **Кому:** Руслан (→ продуктовый по правам)

## Суть

Добавить новые права доступа в Web для следующих сущностей:

| Право | Тип | Описание |
|-------|-----|----------|
| **Подсказки (Hints)** | Кастомное | Доступ к разделу «Подсказки» и управлению ими |
| **Экран покупателя (Customer Screen)** | Автоматическое (App CS) | Доступ к разделу «Экран покупателя» |
| **Доска меню (Menu Boards)** | Автоматическое (App MenuBoard) | Доступ к разделу «Доска меню» |

## Детали

1. **App Arrivals** и **App Digital Voice** — уже зарегистрированы на верхнем уровне
2. Для них добавляется **папка** (группировка прав)
3. Логика: галочка «Активно» → раздел доступен; не активна → раздел скрыт

## Технические детали (от Ромы)

- Права, создаваемые автоматически для каждого приложения: экран покупателя, электронная очередь, звуки
- **Подсказки** — кастомное право (отдельного приложения нет), привязывается к конкретному приложению
- В коде закрываются методы и роуты этими правами

## Что нужно

1. Зарегистрировать новые права в системе прав Web
2. Название менюбордов в интерфейсе — **«Доска меню»** (на русском)
