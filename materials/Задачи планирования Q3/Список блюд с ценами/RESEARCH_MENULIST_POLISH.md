### 1.2. Проблемы (найдено)

| # | Проблема | Серьёзность | Где |

| 1 | Иконка 32×32 — мелковато для canvas | 🟡 | CSS `.ml-icon` |
| 2 | Разделитель строк жёстко `1px solid #eee` — не настраивается | 🟡 | CSS `.ml-row` |

| 6 | Иконка не настраивается по размеру | 🟢 | Тип / инспектор |
| 7 | Нет выбора стиля разделителя (сплошной/пунктир/нет) | 🟢 | Тип / инспектор |
| 8 | Нет выравнивания цены (право/лево) | 🟢 | Тип / инспектор |
| 9 | Нет настройки формата цены (символ валюты, позиция) | 🟢 | Тип / инспектор |

## 2. Сравнение со спецификацией (раздел 7.2)

Целевая структура строки из спеки:

```
┌──────────────────────────────────────────────────────┐
│ [icon]  Cappuccino Scuro                    450 ₽   │
│         Модификаторы: корица, сливки        200 мл   │
│         Размеры: S, M, L                             │
│         Описание: Классический итальянский...         │
│         ⚠ Молоко, Кофеин  |  120 ккал  Б:8 Ж:5 У:10 │
└──────────────────────────────────────────────────────┘
```

**Что совпадает:** базовая структура (иконка слева, название+модификаторы+размеры+описание в центре, цена+граммовка справа).

**Что отличается:**
- Иконка в спеке: 40×40, у нас: 32×32
- В спеке нет префикса «Модификаторы:» — просто перечисление через запятую
- В спеке размеры показаны без цен: «S, M, L», у нас: «S (220₽) · M (280₽)»

## 3. Предложения по улучшению

### 3.1. HIGH — обязательно исправить

#### А. Настраиваемый размер иконки
- Добавить `iconSize?: number` (по умолч. 40)
- Инспектор: поле «Размер иконки (px)» в секции «Отображение»
- Canvas: `[style.width.px]`, `[style.height.px]` на `.ml-icon`

#### Б. Настраиваемый разделитель строк
- Добавить `rowBorderStyle?: 'solid' | 'dashed' | 'none'` (по умолч. `'solid'`)
- Добавить `rowBorderColor?: string` (по умолч. `'#eeeeee'`)
- Инспектор: select «Стиль разделителя» + color picker «Цвет разделителя» в секции «Настройки таблицы»
- Canvas: `[style.border-bottom-style]`, `[style.border-bottom-color]`

#### В. Показ/скрытие граммовки
- Добавить `showWeight?: boolean` (по умолч. `true`)
- Инспектор: чекбокс «Показывать граммовку» в секции «Отображение»

#### Г. Формат цены
- Добавить `currencySymbol?: string` (по умолч. `'₽'`)  
- Добавить `currencyPosition?: 'before' | 'after'` (по умолч. `'after'`)
- Инспектор: поле «Символ валюты» + select «Позиция валюты»
- Canvas: форматировать цену соответственно

### 3.2. MEDIUM — желательно

#### Д. Лейбл модификаторов
- Добавить `showModifiersLabel?: boolean` (по умолч. `true`)
- Когда выкл — модификаторы показываются просто через запятую, без префикса
- Инспектор: чекбокс «Показывать заголовок "Модификаторы:"» в секции «Отображение»

#### Е. Цена для размеров в canvas
- Сейчас в canvas размеры показываются как «S (220₽) · M (280₽)»
- Добавить `showSizePrice?: boolean` (по умолч. `true`)
- Когда выкл — «S · M · L»

#### Ж. Формат размеров (цены внутри)
- Добавить `showSizesInline?: boolean` (по умолч. `true`)
- Когда выкл — размеры не показываются (только выбранный размер)

### 3.3. LOW — nice to have

#### З. Выравнивание цены
- Добавить `priceAlign?: 'right' | 'left'` (по умолч. `'right'`)
- Инспектор: select в секции «Шрифт цены»

#### И. Отступ иконки
- Добавить `iconPadding?: number` (по умолч. `0`)
- CSS: padding на `.ml-icon`

## 4. Что НЕ надо менять

| Аспект | Почему |
|--------|--------|
| Префикс «Размеры:» | OK — помогает отличить размеры от модификаторов |
| Префикс «Модификаторы:» (по умолчанию) | OK — Руслан на встрече не возражал |
| Структура строки (иконка/текст/цена) | Соответствует спеке |
| Цвета по умолчанию | Соответствуют Material Design |
