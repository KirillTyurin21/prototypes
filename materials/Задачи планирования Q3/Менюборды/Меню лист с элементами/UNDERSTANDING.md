# Понимание задачи: Прототип «MenuBoard — Меню-лист с ценами» (Задача 7)

Дата: 2026-06-10

---

## 1. Что нужно сделать (кратко)

В разделе **«Доска меню»** добавить новый элемент темы **«Меню-лист» (MenuList)** и новый контрол **«Меню-борд»**. MenuList — таблица блюд с ценами из внешнего меню. Контрол меню-борда — карточки блюд внутри ControlArea. Источник данных — внешнее меню iiko (мок-данные в прототипе).

---

## 2. Анализ спецификации

### 2.1. Суть задачи

MenuBoard должен уметь показывать блюда с ценами как электронное меню. Два новых UI-элемента:

| # | Элемент | Тип | Где размещается | Визуал |
|---|---------|-----|----------------|--------|
| 1 | **Меню-лист** (MenuList) | Элемент темы (`type: "MenuList"`) | `screens[].elements[]` | Таблица строк: иконка + название + модификаторы + размеры + цена |
| 2 | **Контрол меню-борда** | Контрол внутри ControlArea | Внутри `area`-элемента | Карточки: изображение + название + цена |

### 2.2. Меню-лист — ключевые характеристики

- **Самостоятельный элемент темы** (НЕ контрол внутри ControlArea)
- Отображает список блюд строками: одна строка = одно блюдо
- Источник: внешнее меню iiko (в прототипе — мок-категории и блюда)
- Выбор блюд: модальное окно с деревом категорий + чекбоксами
- Можно добавить несколько MenuList на тему
- Без пагинации в v1 — блюда обрезаются если не помещаются

### 2.3. Структура строки MenuList (из раздела 7.2)

```
┌──────────────────────────────────────────────────────┐
│ [icon]  Cappuccino Scuro                    450 ₽   │
│         Модификаторы: корица, сливки        200 мл   │
│         Размеры: S, M, L                             │
│         Описание: Классический итальянский...         │
│         ⚠ Молоко, Кофеин  |  120 ккал  Б:8 Ж:5 У:10 │
└──────────────────────────────────────────────────────┘
```

Колонки: Иконка (40×40) | Название + модификаторы + размеры | Цена + граммовка

### 2.4. Свойства MenuList в конструкторе (раздел 6.2)

| Вкладка | Свойства |
|---------|----------|
| **Данные** | Кнопка «Выбрать блюда» → модальное окно, список выбранных блюд с удалением |
| **Макет** | X, Y, Ширина, Высота, Слой, Цвет фона, Прозрачность, Отступ, Поворот |
| **Граница** | Цвет, толщина, стиль, скругление |
| **Настройки таблицы** | Чередование строк, Отступ строк, Высота строки |
| **Подсветка строк** | Цвет подсветки |
| **Шрифт названия** | Размер, семейство, цвет, жирность, курсив |
| **Шрифт цены** | Размер, семейство, цвет, жирность, курсив |
| **Шрифт модификаторов** | Размер, семейство, цвет |
| **Шрифт описания** | Размер, семейство, цвет |
| **Отображение** | Показывать иконки, описание, аллергены, КБЖУ |

### 2.5. Модальное окно выбора блюд (раздел 7.4)

- Заголовок: «Выбор блюд из внешнего меню»
- Левая панель (30%): дерево категорий
- Правая панель (70%): список блюд с чекбоксами
- Поиск по названию
- Счётчик «Выбрано N блюд»
- Кнопки «Готово» / «Отмена»

### 2.6. Контрол меню-борда (раздел 6.3)

- Размещается внутри ControlArea
- Карточки блюд: изображение + название + цена
- Размер карточки настраивается (по умолчанию 150×200)
- Направление заполнения: слева направо или сверху вниз

### 2.7. Что НЕ входит в прототип

- Backend API (endpoint `/api/menu-board/external-menu/`)
- Плагин и menuBoardScreen.exe
- Реальное кэширование external_menu_cache
- Интеграция с реальным внешним меню iiko

## 6. Ожидаемый результат

В конструкторе темы «Доска меню»:
1. В палитре «Добавить элемент» → новый пункт «Меню-лист» (первым)
2. На canvas — прямоугольник со списком выбранных блюд
3. В панели свойств — кнопка «Выбрать блюда» → модальное окно с категориями и чекбоксами
4. После выбора — в canvas отображаются названия и цены выбранных блюд
5. Можно добавить несколько MenuList на тему
