# Исходные материалы для прототипов

Эта папка содержит все исходные материалы для работы над прототипами: скриншоты iikoWeb, техническое задание, макеты, документы и другие файлы.

## Структура

Каждый прототип имеет свою подпапку с порядковым номером:

```
materials/
├── Прототип_1/    — материалы для первого прототипа
│   ├── ТЗ.docx
│   ├── скриншоты/
│   └── макеты/
├── Прототип_2/    — материалы для второго прототипа
└── ...
```

## Как использовать

1. Создай новую папку `Прототип_N` (где N — следующий номер)
2. Помести туда все исходные файлы:
   - Техническое задание
   - Скриншоты интерфейса iikoWeb
   - Макеты Figma/Sketch (экспорт)
   - Документацию API (если нужно)
   - Примеры данных
3. При обращении к Copilot ссылайся на эти файлы, чтобы создать прототип

## Примеры содержимого папки прототипа

```
Прототип_1/
├── ТЗ.md                    — описание задачи
├── screenshots/             — скриншоты iikoWeb
│   ├── список.png
│   ├── форма-редактирования.png
│   └── модальное-окно.png
├── mockups/                 — макеты (если есть)
└── data-examples.json       — примеры данных (опционально)
```

> **Совет**: сохраняй скриншоты в PNG с понятными названиями (не `Screenshot_123.png`, а `экран-списка-товаров.png`).
